# walkthrough — RadioCode.Space (v1 доводка)

Артефакт ИИ-агента по наряду-заданию Архитектора от 2026-07-19.

## Что и зачем меняется

| Область | Было | Стало | Файлы |
|---|---|---|---|
| Ротация плейлиста | наивный `(i+1)%N` и random != current — версии одной композиции могли идти подряд | эпохи с зазором ≥⌊N/2⌋ + round-robin версий, персист в localStorage | `src/lib/rotation.ts`, `src/stores/playerStore.ts` |
| Битрейт на сайте | «320 KBPS HQ», бейджи 320/256 | фактические VBR-значения из ffprobe/mutagen | `src/lib/stations.ts`, `src/app/page.tsx`, `src/app/layout.tsx` |
| Битые названия | 4 названия с mojibake-символами | восстановлены | `src/lib/stations.ts` |

## Ключевые инварианты ротации (проверены автотестом)
- В пределах эпохи каждая композиция играет ровно один раз.
- Между двумя проигрываниями одной композиции — не менее ⌊N/2⌋ других композиций.
- Версии композиции идут round-robin (не предсказуемо v1→v2→v3).
- Состояние переживает перезагрузку страницы.
- Все 596 файлов сохранены; версии не удаляются и не объединяются.

## Артефакты
- `reports/compositions.json` — группировка файлов по композициям (160 групп).
- `reports/suspicious_merges.json` — проверка склеек (3 ложных флага из-за битых названий).
- `reports/bitrate.csv` — измеренный битрейт всех 596 файлов.
- `scripts/test-rotation.ts` — автотест ротации (5000 итераций/станцию).

## Запуск автотеста
```
node --experimental-strip-types scripts/test-rotation.ts
```

## Деплой
Проект НЕ git-интегрирован с Vercel. Деплой ТОЛЬКО через CLI из папки:
```
cd radiocode-space && vercel --prod --yes
```
Git push в `main` — только резервная копия на GitHub, деплой НЕ вызывает.
