
## 2026-07-20 — Нормализация громкости (реально «на одном уровне»)

В прямом режиме AGC через Web Audio невозможен (вернул бы зависание). Реализовал per-track нормализацию БЕЗ Web Audio и БЕЗ перекодирования:
- Замер интегральной громкости всех 596 файлов через Python `miniaudio` (самодостаточный декодер MP3, без ffmpeg) → `reports/loudness.json`. Распределение: медиана −16.27 LUFS, разброс **6.31 dB** (−19.82…−13.51).
- Коэффициент на трек: attenuation-only к цели **−17.3 LUFS** (`gain = min(1, 10^((target−lufs)/20))`, клэмп [0.3,1]). Громкие приглушаются к цели, тихие остаются (буст без Web Audio невозможен). Разброс после: **2.53 dB** (533/596 приглушены, медиана точно −17.30).
- `gain` вписан в каждый трек `stations.ts`; движок применяет `deck.el.volume = userVolume · masterFactor · gain`. Debug-геттер `window.__radioEngine.debug()`.
- **Проверено вживую** (встроенный Chromium): 5 треков — разные gain (0.797/0.911/0.925/…), фактическая громкость дека = userVol·gain (ratioOK на всех), играет непрерывно.
