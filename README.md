# RadioCode.Space — CODE Radio

RadioCode.Space is an immersive, zero-cost audio-visual web sanctuary. We deliver a seamless, high-performance HTML5 player wrapped in premium glassmorphism aesthetics. Designed for creators, it supports dynamic track folders, 60fps reactive animations, and uninterrupted playback without any expensive backend streaming infrastructure.

Живое радио экосистемы **CODE (Code of Digital Eternity)** — иммерсивный веб-плеер с визуализациями, эквалайзером, плейлистами и эффектами.

- Продакшн: https://radiocode.space
- Часть экосистемы: [codeofdigitaleternity.com](https://www.codeofdigitaleternity.com) · [aifa.digital](https://aifa.digital) · [aifa.works](https://aifa.works)

## Стек

Next.js (App Router, standalone) · React · Tailwind CSS · shadcn/ui · Web Audio API.

## Запуск

```bash
npm install        # или bun install
npm run dev        # http://localhost:3000
npm run build && npm start
```

Сайт сгенерирован в Z.ai и перенесён 1:1; доработки ведутся в этом репозитории.
